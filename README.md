### Private product work

- **Nexxus**  
  Real-time WISP monitoring UI built to keep 200+ devices visible on one screen. The key win was identifying the real bottleneck in state flow and render churn, then getting the map to roughly 60 FPS without forcing a premature renderer rewrite.

- **Fast Or Nothing Admin Platform**  
  Angular 21 redesign and rebuild of a broken legacy admin. The work was about product recovery, stronger UX, cleaner structure, and a foundation that can actually scale.

- **NetGest**  
  WISP operations CRM covering customers, billing, ticketing, technical visits, RBAC, notifications, and monitoring inside one coherent internal platform.

- **Risk Evaluation Platform**  
  Industrial risk evaluation product with structured workflows, protected flows, and report generation built for operational use, not generic form handling.

- **Crypto Fraud Investigation Workflow**  
  AI-assisted operational pipeline built with AdonisJS, React, and n8n, producing Excel and DOCX outputs for downstream investigation work.
